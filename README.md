# Using river-diffusion 5 - map

Enter the folder `tmp_species_B` through for example `cd ~/ug4/apps/krebse/tmp_species_B`.

Run: ```ugshell -ex krebse/river-diffusion_5_map.lua```

Run and clear: ```rm *.vtu && ugshell -ex krebse/river-diffusion_5_map.lua```

Run and clear and log: ```rm *.vtu && ugshell -ex krebse/river-diffusion_5_map.lua > text.log```

The geometry used is `simple-river-y2-meters.ugx`
